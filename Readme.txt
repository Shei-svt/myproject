Este software en Python optimiza el pago de órdenes de compra según el monto total, distribuyéndolo entre inversión propia, préstamo bancario y crédito al fabricante (con un 20% de interés en lo financiado).  

Cuenta con control de versiones en Git, un *main.py* para los cálculos, un *readme.txt* con la documentación y una carpeta *docs/* para referencias. Su desarrollo modular y buenas prácticas garantizan mantenimiento sencillo. Disponible en GitHub, facilita la gestión financiera de compras.
